# Phase 2 Ocean Explorer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn Ocean Jukebox from a refreshed archive jukebox into a richer ocean-sound explorer with shareable tracks, a detail drawer, cleaner catalog grouping, a sanctuary map, live/near-live sources, spectrogram support, and stronger QA.

**Architecture:** Keep the app static and dependency-light. Move reusable pure behavior into `app-core.mjs`, keep browser DOM wiring in `app.js`, store generated archive tracks in `sounds.json`/`sounds.js`, add separate static source files for sanctuary map metadata and live source metadata, and generate spectrogram image assets with an optional Node script.

**Tech Stack:** Vanilla HTML/CSS/JS, Node.js built-in test runner, NOAA SanctSound catalog, optional static SVG/PNG spectrogram artifacts generated by Node/FFmpeg when available, no bundler, no backend.

---

## Scope Check

This phase covers several product surfaces, but they are coupled through one catalog/player state model and should ship in ordered slices. Do not build a backend, authentication, user accounts, analytics, or a full GIS system. Do not present SanctSound archive files as realtime audio.

## File Structure

- Modify `app-core.mjs`: pure state helpers for URL params, track IDs, grouped original/enhanced versions, map filtering, live source status normalization, and spectrogram path selection.
- Modify `tests/app-core.test.mjs`: tests for every pure helper before implementation.
- Modify `scripts/catalog.mjs`: enrich generated catalog with stable IDs, original/enhanced grouping keys, and manual overrides.
- Create `catalog-overrides.json`: hand-curated metadata for odd files and grouping corrections.
- Modify `tests/catalog.test.mjs`: tests for overrides, stable IDs, grouping keys, and compact timestamps.
- Modify `sounds.json` and `sounds.js`: regenerated catalog after generator changes.
- Modify `index.html`: add semantic containers for tabs, detail drawer, map panel, live panel, spectrogram panel, and share controls.
- Modify `app.js`: browser wiring for URL sync, drawer, copy link, map interactions, tabs, live panel, spectrogram display, and accessibility states.
- Create `live-sources.js`: browser-loaded metadata for live/near-live source cards.
- Create `sanctuaries.js`: sanctuary coordinates and display metadata.
- Create `scripts/generate-spectrograms.mjs`: optional local generator for static spectrogram images.
- Create `spectrograms/.gitkeep`: keeps output directory in git.
- Modify `README.md`: document phase-two features, live-source caveats, spectrogram generation, and verification commands.

```js
  const params = new URLSearchParams();
  if (route.tab && route.tab !== 'archive') params.set('tab', route.tab);
  if (route.track) params.set('track', route.track);
  if (route.category && route.category !== 'all') params.set('category', route.category);

  if (route.query) params.set('q', route.query);
  if (route.sort && route.sort !== 'curated') params.set('sort', route.sort);
  const query = params.toString();
  return query ? `?${query}` : '';
```

```js
    query: params.get('q') || '',
    sort: params.get('sort') || 'curated',
    tab: params.get('tab') || 'archive',
  };

function serializeRoute(route) {
  const params = new URLSearchParams();
  if (route.tab && route.tab !== 'archive') params.set('tab', route.tab);
  if (route.track) params.set('track', route.track);
  if (route.category && route.category !== 'all') params.set('category', route.category);
  if (route.sanctuary && route.sanctuary !== 'all') params.set('sanctuary', route.sanctuary);
  if (route.query) params.set('q', route.query);
  if (route.sort && route.sort !== 'curated') params.set('sort', route.sort);
  const query = params.toString();
  return query ? `?${query}` : '';
}

function syncUrl() {
  const track = state.tracks[state.currentIndex];
  const query = serializeRoute({
    track: track ? trackId(track) : '',
    category: state.category,
    sanctuary: state.sanctuary,
    query: state.query,
    sort: state.sort,
    tab: state.activeTab || 'archive',
  });
  window.history.replaceState(null, '', `${window.location.pathname}${query}${window.location.hash}`);
}

function applyInitialRoute() {
  const route = parseRoute();
  state.category = route.category;
  state.sanctuary = route.sanctuary;
  state.query = route.query;
  state.sort = route.sort;
  state.activeTab = route.tab;
  const routedIndex = state.tracks.findIndex(track => trackId(track) === route.track);
  if (routedIndex >= 0) state.currentIndex = routedIndex;
}
```

```js
function variantForFilename(filename) {
  return /(?:Speed|gain|dB|dBGain|dBgain)/i.test(filename) ? 'enhanced' : 'original';
}

export function buildGroupKey(metadata) {
  const cleanedSlug = String(metadata.soundSlug || metadata.filename || 'unknown')
    .replace(/_(?:\d+xSpeed|[\ddB]+gain|[\ddB]+Gain).*$/i, '')
    .replace(/[^a-z0-9]+/gi, '-')
    .replace(/^-|-$/g, '')
    .toLowerCase();
  if (metadata.site && metadata.deployment) {
    return `${metadata.site}-${metadata.deployment}-${cleanedSlug}`;
  }
  return cleanedSlug || 'unknown';
}

export function applyOverrides(tracks, overrides = {}) {
  return tracks.map(track => ({
    ...track,
    ...(overrides[track.filename] || {}),
  }));
}
```

```js
export function buildVariantGroups(tracks) {
  const groups = new Map();
  tracks.forEach(track => {
    const key = track.groupKey || trackId(track);
    if (!groups.has(key)) groups.set(key, { original: null, enhanced: [] });
    const group = groups.get(key);
    if (track.variant === 'enhanced') {
      group.enhanced.push(track);
    } else if (!group.original) {
      group.original = track;
    } else {
      group.enhanced.push(track);
    }
  });
  return groups;
}
```

```js
export function buildMapPins(sanctuaries, tracks, activeSanctuary = 'all') {
  const counts = new Map();
  tracks.forEach(track => {
    counts.set(track.sanctuary, (counts.get(track.sanctuary) || 0) + 1);
  });
  return sanctuaries.map(sanctuary => ({
    ...sanctuary,
    count: counts.get(sanctuary.name) || 0,
    active: sanctuary.name === activeSanctuary,
  }));
}
```

```js
function renderMap() {
  els.mapCanvas.innerHTML = '';
  const pins = state.sanctuaries.map(sanctuary => ({
    ...sanctuary,
    count: state.tracks.filter(track => track.sanctuary === sanctuary.name).length,
    active: state.sanctuary === sanctuary.name || state.tracks[state.currentIndex]?.sanctuary === sanctuary.name,
  }));
  pins.forEach(pin => {

    button.type = 'button';
    button.className = `map-pin${pin.active ? ' active' : ''}`;
    button.style.left = `${lonToX(pin.lon)}%`;
    button.style.top = `${latToY(pin.lat)}%`;
    button.setAttribute('aria-label', `${pin.name}, ${formatCount(pin.count)}`);
    button.innerHTML = `<i class="ti ti-map-pin" aria-hidden="true"></i>`;
    button.addEventListener('click', () => {

      els.sanctuarySelect.value = pin.name;
      recomputeVisible();
      renderTrackList();
```

```js
import { mkdir } from 'node:fs/promises';
import { spawnSync } from 'node:child_process';

const [, , filename] = process.argv;

if (!filename) {
  console.error('Usage: node scripts/generate-spectrograms.mjs <filename-from-sounds-json>');
  process.exit(1);
}

const catalog = await import('../sounds.json', { with: { type: 'json' } });
const track = catalog.default.tracks.find(item => item.filename === filename);

if (!track) {
  console.error(`No track found for ${filename}`);
  process.exit(1);
}

const id = track.filename.replace(/\.[^.]+$/, '');
const inputUrl = track.url;
const output = `spectrograms/${id}.png`;

await mkdir('spectrograms', { recursive: true });

const ffmpeg = spawnSync('ffmpeg', [
  '-y',
  '-i', inputUrl,
  '-lavfi', 'showspectrumpic=s=1200x520:legend=disabled:color=viridis',
  output,
], { stdio: 'inherit' });

if (ffmpeg.status !== 0) {
  console.error('ffmpeg failed. Install ffmpeg or generate this asset manually.');
  process.exit(ffmpeg.status || 1);
}

console.log(`Wrote ${output}`);
```
